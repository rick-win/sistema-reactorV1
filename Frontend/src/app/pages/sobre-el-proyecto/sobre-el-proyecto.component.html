    • LDP DIGITALIZACIÓN DEL PROCESO DE PIRÓLISIS


    Debido a que la producción de plástico ha aumentado exponencialmente afectando nuestro planeta entero, continuamente se busca tecnologias que aprovechen estos recursos y sean transformados para ser reutilizados y asi no se genere mas desechos en el medio ambiente.

    Una de las alternativas de aprovechamiento, el reciclaje químico. Este sobresale por presentar un cambio físico a uno químico degradando los polímeros a compuestos químicos base y combustible. Este tipo de reciclaje presenta cuatro métodos tecnológicos estos son pirólisis, hidrogenación, gasificación, metanólisis. Para este proyecto, se selecciono el proceso de pirólisis, el cual, es un proceso inverso de ingeniería que permite obtener los elementos base con los que fue construido el producto de plástico en presencia de temperaturas altas y muy bajos o ausencia de niveles de oxígeno.

    Para esta planta piloto, se implemento un prototipo de sistema de información haciendo uso de diferentes tecnologías de la información y la comunicación. Esta implementacion fue realizada gracias a la colaboracion de diferentes entidades que se puede detallar en los siguientes menus.
